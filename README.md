 
# 🏦 Bank Data Analysis Dashboard 📊
 
This project presents an **interactive dashboard** visualizing key metrics from a bank's customer transaction dataset. The dashboard is designed in **Power BI** and aims to uncover insights such as top customers, transaction patterns, movement types, gender distribution, and more.

---

## 📌 Overview

- **Total Balance**: 💰 177.08M  
- **Total Customers**: 👥 12.04K  
- **Tool Used**: Power BI   
- **Purpose**: To analyze customer behavior and transaction trends in the banking system for improved decision-making.

## 📅 Filters Used in Dashboard

- **Month Filter**: August, September, October, November
- **Day Filter**: All seven days of the week available
- This enables customized and dynamic analysis for specific periods.

---

## 🧠 Key Insights

- Target **Tonya**, **Susan**, and **Tiffany** for upselling banking products based on high transaction counts.
- Focus on **Susan**, **Tyler**, and **Tim** for premium customer services due to high spending.
- Prioritize improvements in transaction processing to move more from "Authorized" to "Posted".
- Run targeted campaigns for **20–40 age group** across both genders.
- Analyze why debit usage is so low and improve offerings (e.g., UPI cashback or debit card benefits).
